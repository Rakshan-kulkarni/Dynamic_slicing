# Dynamic_slicing
# Dynamic Network Slicing Simulation

This project simulates **dynamic network slicing** using Python.

The simulation models multiple clients connecting to base stations and shows how network resources are allocated dynamically.

## Technologies Used

- Python
- NumPy
- Matplotlib
- Jupyter Notebook

## How to Run

1. Clone the repository

git clone https://github.com/YOUR_USERNAME/dynamic-network-slicing-simulation.git

2. Install requirements

pip install numpy matplotlib jupyter

3. Run the notebook

Open:

dynamic_slicing.ipynb

using Jupyter Notebook.
